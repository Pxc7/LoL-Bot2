# Installation

## Termux
```cmd
> pkg update && pkg upgrade
> pkg install git -y
> pkg install nodejs -y
> pkg install ffmpeg -y
```

## Cloning 
```cmd
> git clone https://github.com/Pxc7/LoL-Bot2
> cd LoL-Bot2
```

## Install 
```cmd
> npm i
```

## Run the LoL-Bot2
```cmd
> npm start
```
